# COLORS Lab Web Application

## Project Description
COLORS Lab is a web application designed as an exercise in UCF's COP4331 Lab. 
Users can create and account to store colors of their choosing. 
The purpose of this lab is to teach the basics of frontend and backend development using a LAMP stack. 


## Technologies Used
- **Linux / Apache / MySQL / PHP** (LAMP stack)
- **HTML5, CSS3, JavaScript** (frontend)
- **PHP** (backend API logic)

## Setup Instrucions
1. Clone this repository: git clone https://github.com/YOUR_USERNAME/colors-lab-cop4331.git
2. Open the project folder: cd colors-lab-cop4331
3. Make sure you have a local LAMP Stack installed (Linux, Apache, MySQL, PHP)
4. Put the project in your web server root (/var/www/html/)

## Run / Access Application
1. Start your web and database servers
2. go to http://localhost/colors-lab-cop4331/public/index.html in your web browser

## Assumptions and Limitations
- Assumes user has LAMP installed
- Users and accounts only stored for demonstration, high level security is lacking
